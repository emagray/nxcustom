# Contributing to NXcustom

Thank you for your interest in improving NXcustom!

## How to Contribute

### Reporting Issues

1. Check [existing issues](https://github.com/emagray/nxcustom/issues)
2. Provide clear description
3. Include reproduction steps
4. Attach relevant files (anonymized)

### Suggesting Improvements

- Create an issue with `[Enhancement]` label
- Explain use case and benefit
- Provide examples if possible

### Documentation Improvements

- Fix typos, clarify explanations
- Add examples and use cases
- Improve organization and formatting

### Code/Script Contributions

NXcustom welcomes contributions:
- Bug fixes
- Performance improvements
- New features
- Documentation

## Guidelines

- Test thoroughly before submitting
- Follow existing code style
- Include documentation
- Provide clear commit messages
- Reference related issues

## Licensing

By contributing, you agree to license your contributions under the same terms as the project.
